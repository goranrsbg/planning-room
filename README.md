## Planning Room

Simple scrum poker like app.

Using \
[Jakarta EE 10](https://jakarta.ee/release/10)

### Run

> mvn clean install

> cd room

> mvn cargo:run

- then

```
http://localhost:8080/planning-room-web
https://localhost:8181/planning-room-web
```

#### Docker cli

> ./run.sh

- then

``` 
http://localhost:8080/planning-room-web
https://localhost:8181/planning-room-web
```
